# Steps to Set up Active Jobs on the Plugin Feeds

**Note:-** Node Uptime is of utmost importance when you have a live job.

---

## üîó GitHub Repository

[https://github.com/GoPlugin/datafeed_jobs_StandardNodes](https://github.com/GoPlugin/datafeed_jobs_StandardNodes)

This GitHub repository contains pre-defined job templates for various data feed pairs (e.g., ATOM/USDT, DOGE/USDT, CAKE/USDT).

Each folder includes job definitions using different price data sources - CoinMarketCap, Bitrue and CryptoCompare.

These definitions are designed to be directly used in the Plugin node, making it easier to set up and run live data feeds for your chosen crypto pair.

---

## Setup Instructions

Login in to your Plugin Node user interface.

Set up a new job using the definition provided in the GitHub repository, specific to the price pair and data source assigned to you.

Navigate to the ‚ÄúJobs‚Äù section.

Click on the ‚ÄòNew Job‚Äô button to add a job definition for each price pair.

<img width="1600" height="102" alt="image" src="https://github.com/user-attachments/assets/8694bb58-8232-4045-8263-7905ba1336e8" />


As mentioned above, in the GitHub repository, each folder contains job definitions with different sources. Copy the job definition that matches the price pair and source assigned to you.

Paste the job definition and click on the ‚ÄòCreate New Job‚Äô button.
